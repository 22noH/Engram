# Quiet Library — Engram 전 표면 디자인 시스템

날짜: 2026-07-23 · 상태: 방향 확정(목업 3안 중 A 선택·다크/위키 장면 승인)

## 콘셉트

"지식 서재." Engram의 본질(대화가 지식이 되는 곳)을 타이포그래피로 표현한다 — **말(대화)은 산세리프, 기록(지식)은 세리프**. 차분한 종이톤 바탕, 깊은 청록 잉크 액센트, 절제된 장식. 다크 모드는 검정이 아니라 따뜻한 흑연(서재의 밤).

적용 표면: ① 앱 렌더러 전체(채팅·사이드바·위키·스레드·모달·팔레트·질문카드) ② 데스크톱 설정창(settings.html) ③ 웹 관리 콘솔(/admin). 세 표면이 같은 토큰을 쓴다.

### 타이포그래피

- `--font-sans`: `'Inter', system-ui, -apple-system, 'Segoe UI', sans-serif` — 대화·UI 전부. 번들(OFL).
- `--font-serif`: `'Newsreader', 'Noto Serif KR', 'Batang', Georgia, serif` — **지식 표면 전용**: 위키 본문·제목, 채팅 속 위키 유래 콘텐츠(compact 요약 앵커·위키 인용 콜아웃), 사이드바 Wiki 항목(이탤릭). Newsreader 번들(OFL), 한글은 시스템 폴백(Noto Serif KR 번들은 용량 트레이드오프로 후속 결정 — 스펙상 폴백 스택만 보장).
- `--font-mono`: 기존 Consolas 스택 유지(코드).
- 스케일: 본문 13px/1.55(채팅)·14.5px/1.75(위키 세리프 본문)·12.5px(컨트롤)·10px 대문자 눈썹(letter-spacing 1.2px, accent 색) — 카테고리·날짜·섹션 라벨용. 위키 제목 21px/600 세리프.

### 형태·간격

- radius: 버블·카드 10px, 입력·버튼 9px, 배지·필 7px, 번호칩 4px.
- 사이드바 활성: `--accent-soft` 배경+우측 2px 액센트 바(기존 패턴 유지). 섹션 라벨은 눈썹 스타일.
- 위키 본문 폭 제한 `max-width: 62ch`(책 같은 읽기 흐름)·버튼류는 산세리프 유지.
- 그림자·그라데이션 금지(기존 플랫 유지). 모션은 최소: 위키 저장 성공 순간 액센트 1회 페이드 정도(후속·비필수).

## 컴포넌트 규칙(대표)

- **메시지**: Engram=`--panel-2`+`--line`, 나=`--me-bg`+`--me-line`(우측 정렬 유지), 타인=`--code-bg`. 타임스탬프 `--faint`.
- **위키 인용/요약(채팅 내)**: 세리프 본문+눈썹 라벨("FROM THE WIKI"/"COMPACTED") — 좌측 3px 액센트 보더.
- **질문 카드(qcard)**: 기존 구조 그대로, 토큰만 재적용(선택 칩=accent·비선택 칩=`--line` 톤). 목업 A-다크 승인 형태.
- **위키 읽기**: 눈썹(카테고리·갱신일)→세리프 제목→세리프 본문(62ch)→산세리프 컨트롤 행(Unpublish는 `--danger`).
- **모달**(Engram 관리·멤버·팔레트): `--panel` 카드+헤어라인, 섹션 라벨은 눈썹.
- **설정창·콘솔**: 동일 토큰의 자체 CSS 적용(콘솔은 관리 화면이라 세리프 미사용 — 눈썹·팔레트만 공유).

## 전역 제약

- **레이아웃 구조 변경 없음**: 기능 배치·기존 클래스명·기능 DOM 유지. 단 **순수 표현용 추가는 허용**(사이드바 섹션 눈썹 라벨·위키 눈썹 줄 등 — 동작·핸들러·기존 테스트 셀렉터에 영향 없는 정적 요소만). 문구 변경 없음. 신규 기능 0.
- 폰트는 로컬 번들(런타임 네트워크 요청 금지 — 데스크톱·콘솔 모두). OFL 라이선스 표기.
- 대비: 본문 텍스트 대비 WCAG AA 이상(라이트 #24292e/#f5f6f4, 다크 #e6e5df/#191b19 — 둘 다 충분). accent 위 글자는 `--accent-text`로만.
- 회귀 0: jsdom 테스트는 클래스명 의존이라 스타일 값 교체로는 불변. 기존 `[hidden]` 등 기능성 CSS 규칙 유지.

## 비목표

- 정보 구조·내비게이션 재설계(별도 작업). 애니메이션 시스템(후속). 사용자 테마 선택 UI(자동 라이트/다크만).

- 콘솔·설정창 스모크: 기존 스모크 green+육안.
