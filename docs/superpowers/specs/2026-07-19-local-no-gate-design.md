# 배포 형태 분리(스탠드얼론/클라이언트/서버) — 설계

날짜: 2026-07-19
상태: 승인됨. (이 파일의 이전 버전 "로컬 무게이트"를 대체 — 조건부 게이트 숨김이 아니라
배포 형태 자체를 가른다. 사용자가 애초에 지적했던 구조.)

## 1. 확정 구조 (사용자 결정)

| 물건 | 화면 | 로그인 |
|---|---|---|
| **데스크톱 앱 (스탠드얼론 = preset 없음)** | 챗봇·코드·위키. **채팅(team) 탭 자체가 없음** | **없음** |
| **데스크톱 앱 (preset 배포 = 설치 때 서버 주소가 박혀 나감)** | + 채팅(team) 탭 | **그 서버의 로그인만** |
| **서버 (별도 배포물)** | 없음(헤드리스 데몬) | — (owner 계정은 CLI 셋업, 클라이언트가 로그인) |

- 판단 로직 없음: **preset.json 유무가 전부.** 어디에 붙을지는 설치/배포 시점에 이미 결정.
- 초대 링크·첫 실행 선택 화면·런타임 추측 — 전부 없음.
- "내 서버 만들기"(owner 셋업 화면)는 데스크톱 앱에서 **삭제**. 서버 셋업은 서버 배포물의
  CLI 절차(기존 setup-code 흐름 재사용) — 서버 배포물 정리는 다음 페이즈.

## 2. 설계

### 2.1 서버 측 — 스탠드얼론 로컬 백엔드의 무인증 (self.adapter + auth-http)

데스크톱이 띄우는 로컬 백엔드는 기본 바인드가 127.0.0.1(기존 chat.config 기본값)이라
외부 접속이 물리적으로 불가 — 그래서 무인증이 안전하다.

- ws: **계정 0개(미설정) AND 소켓이 루프백**이면 인증 프레임 없이 통과 — 기존 brain
  모드(무인증)와 동일한 권한 경로 재사용(새 분기 발명 금지). 판정은 요청 시점
  `accounts.count()` 재조회(캐시 금지).
- `/auth/status` 응답에 `localFree: boolean` 추가(계정 0 + 요청 루프백). 루프백 판정은
  소켓 remoteAddress만(헤더 신뢰 금지 — 8c-2 isLoopback 관성).
- 비루프백·계정 있는 서버·brain 모드·setup-code API 전부 현행 무변경.
- 엣지: 계정이 이미 만들어진 로컬 백엔드(레거시)는 현행 게이트 유지 — 본 사용자 해당
  없음(계정 미생성), 문서화만.

### 2.2 렌더러 — 스탠드얼론/클라이언트 분리

- **채팅(team) 탭**: 현재 `TEAM_CHAT=true` 하드코딩(Phase 14) → **preset 존재 시에만
  표시**(config.ts의 PRESET 유무). 스탠드얼론=탭 자체가 안 보임.
- **게이트**: preset 연결(원격 서버)에만 적용 — 로컬 연결(local)은 `/auth/status`의
  `localFree`로 게이트 생략. preset 서버의 로그인/가입 화면은 유지(어느 서버인지 이름
  명시, 세션은 연결별 저장 — 기존 그대로).
- **"내 서버 만들기"(setup 뷰) 삭제**: LoginGate에서 `!status.configured` 셋업 폼 제거 —
  미설정+원격은 "서버가 아직 준비되지 않았어요" 안내로 대체(원격 owner 셋업은 서버
  CLI 소관).
- **CSS 버그 픽스**: `input[type=text]`에만 적용되던 입력칸 스타일을
  `input[type=password]`에도(게이트 폼 아이디/비번 칸 모양 불일치).

### 2.3 서버 배포물 (다음 페이즈 — 이번 비범위)

- 헤드리스 데몬(`engram-server`): GUI 없음, 설정 파일+CLI(포트·바인드·서버명·owner
  셋업=기존 setup-code의 CLI화), preset.json 생성 도구(클라이언트 배포용). 백로그 등재.

## 3. 테스트

- auth-http: 미설정+루프백=localFree true·계정 생기면 false·비루프백=false.
- self.adapter: 미설정+루프백 ws 무인증 통과·계정 생성 즉시 재가동·비루프백 현행·brain
  모드 회귀 0.
- 렌더러: preset 없으면 team 탭 미렌더·있으면 렌더, localFree=게이트 생략, setup 뷰
  부재(미설정 원격=안내 문구), 기존 로그인 폼 회귀.
- 실스모크: 실기기 — 스탠드얼론 부팅=게이트 없음·채팅 탭 없음·챗봇/위키 동작,
  ws 무인증 실왕복, 계정 만들면 게이트 재가동(임시 데이터로).

## 4. 비범위
