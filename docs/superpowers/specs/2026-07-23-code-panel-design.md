# 코드 패널 — 코드 채널 우측 터미널·미리보기·Diff

날짜: 2026-07-23 · 상태: 설계 확정(목업 B+라이트 터미널 수정본 사용자 승인)

## 목적

코드 채널에서 Claude Code 데스크톱처럼 우측 분할 패널로 **진짜 터미널·미리보기·git diff**를 제공한다.

## 범위 (사용자 확정)

- **코드 채널에서만** 상단바 우측 아이콘 3개(터미널·미리보기·Diff) 활성. 다른 모드에선 미노출.
- **데스크톱 전용**: 실제 셸·로컬 레포 접근이라 `window.engramDesktop` 브리지가 있는 데스크톱 앱에서만. 원격 preset 클라이언트에는 노출하지 않음(서버 셸 원격 노출은 별개 보안 문제 — 비목표).

## 패널 공통

- 우측 분할 패널, 탭(터미널/Preview/Diff·변경수 배지)+닫기. 폭 드래그 조절(로컬 저장·퍼시스트), 열림 상태 채널별 기억(로컬).
- Quiet Library 토큰 추종 — **터미널 배경도 앱 테마**(라이트=종이톤·다크=흑연, 검정 고정 아님 — 사용자 확정).

## 터미널

- **OS 기본 셸**: 윈도우=PowerShell, 맥=zsh, 리눅스=`$SHELL`(없으면 bash). 탭 라벨은 실제 셸 이름 표시.
- 스택: node-pty(메인 프로세스)+xterm.js(렌더러) — pty 스폰·입출력은 IPC로만(렌더러에 노드 API 미노출 유지). node-pty는 네이티브 모듈 — electron-rebuild/프리빌드 검증 필수(플랜 리스크 항목).
- cwd=채널 `repoPath`. 채널당 1세션 — 패널 닫기=숨김(세션 유지), 채널 삭제·앱 종료 시 kill(고아 방지, before-quit 정리).
- 보안: 로컬 사용자의 자기 권한 셸(앱 권한 그대로). 두뇌·원격과 무관(두뇌 코딩은 기존 cmdGuard 경로 유지 — 이 터미널과 분리).

## 미리보기(Preview)

- URL 바+임베드 브라우저(`<webview>`/iframe — Electron 보안 설정 확인해 선택, nodeIntegration 금지)+새로고침·외부 브라우저 열기 버튼.
- 빈 상태: URL 입력 안내(예: `http://localhost:5173`). 자동 감지는 비목표(후속).

## Diff

- 대상: 채널 repoPath의 **git 워킹트리 diff**(읽기 전용). 좌측 변경 파일 목록(A/M/D 상태)+우측 선택 파일 unified diff(추가=초록계·삭제=danger계 — 토큰 파생). 탭 배지=변경 파일 수(주기 갱신 또는 탭 열 때 갱신).
- git 실행은 메인 프로세스 IPC(기존 wiki-git 관례처럼 자식 프로세스)·레포 아니면/`git` 없으면 안내 문구.
- 비목표: 스테이징·커밋·되돌리기 등 쓰기 조작(뷰어만).

## 안전선

- 회귀 0: 코드 패널 미사용 시 기존 화면 byte-identical. 채팅 영역 중앙 칼럼과 공존(패널 열리면 남는 폭 안에서 칼럼 유지).
- pty·git·preview 전부 IPC 경유(contextIsolation 유지)·렌더러 직접 노드 접근 금지.
- 터미널 세션·자식 프로세스는 종료 시 전부 정리(supervisor before-quit 관례).

## 비목표

원격 클라이언트 노출·터미널 멀티탭/분할·diff 쓰기 조작·preview 자동 포트 감지·두뇌가 이 터미널을 조작(두뇌는 기존 코딩 경로).
