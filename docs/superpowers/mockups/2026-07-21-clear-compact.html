Engram — /clear · /compact (앱 채팅)

앱 채팅에 대화 정리 두 명령을 추가한다. 실제 renderer/src/theme.css 토큰·클래스로 그렸다(라이트/다크 자동). 채널당 동작이며 위키/RAG 지식은 절대 지우지 않는다.

모달 없이 인-챗 방식(사용자 결정): 슬래시/메뉴로 치면 바로 실행.
/clear = 이 채널 대화 기록 즉시 삭제 (확인창·시스템 메시지 없음 · 요약 안 남김).
/compact = 채널 AI가 요약을 채팅 메시지로 한 번 보여주고 → 위키에 바로 저장 → 기록 정리. 그 요약 메시지가 앵커로 남아 RAG가 읽어 이어감.
실행 위치 = 입력창 슬래시 명령(기존 팔레트에 슬롯 추가) + 채널 ⋯ 메뉴 버튼 2개. 위키/RAG 지식은 절대 안 지움.
① 입력창 슬래시 팔레트 — / 치면 기존 명령들과 함께 /clear·/compact가 뜬다

모달 없이 요약이 채팅 메시지로 한 번 뜨고(무엇을 저장하는지 봄) → 위키에 바로 저장(propose+approve, dedup append) → 옛 기록 정리. 이 요약 메시지가 새 대화의 앵커로 남아 RAG가 읽어 이어감. 슬래시·⋯메뉴 동일.

⑤ 자동 compact — S4 자동 보존이 오래된 대화를 지우기 직전 자동 요약 → 위키 바로 게시 → 프루닝
Engram은 컨텍스트가 "차오르는" 구조가 아니라 항상 최근 6턴 + 위키 RAG만 본다. 그래서 클로드코드식 "꽉 참 게이지"가 아니라, 오래된 대화가 보존 정책에 의해 잊혀지기 직전에 자동 요약해 위키로 보낸다 → "잊되 지식은 안 잃는다". 자동 저장은 continuity를 위해 바로 게시(대기열이면 RAG가 못 읽어 이어짐이 끊김) + auto-compact 태그로 사후 검토.
